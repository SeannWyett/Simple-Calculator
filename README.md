## Simple Web Calculator

A simple web-based calculator built using **HTML**, **CSS**, and **JavaScript**.  
It can perform basic arithmetic operations — **addition**, **subtraction**, **multiplication**, and **division** — all in your browser.

## Features

- Perform basic math operations:
  - ➕ Addition  
  - ➖ Subtraction  
  - ✖️ Multiplication  
  - ➗ Division  
- Clear and delete buttons for easy corrections  
- Responsive and clean UI  
- Lightweight — no frameworks required  

## Technologies Used

**HTML5** – Structure of the calculator  
**CSS3** – Styling and layout design  
**JavaScript (ES6)** – Logic for calculations and button functionality  

## Instructions
1. Click on your desired number
2. Click on your desired operation
3. Click on your second desired number
4. Click on the equal(=) sign
5. View the result
